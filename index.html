I. 컴퓨팅 시스템

개념 1. 네트워크 구성 요소 및 주소 체계

교과서 연계: I단원 네트워크의 이해

네트워크는 다양한 정보를 송수신하기 위해 연결된 통신망을 뜻하며, 다음의 필수 요소들로 이루어집니다.

단말 기기

데이터를 최종 송수신하는 목적지 기기입니다.

• 데스크톱, 노트북, 스마트폰 등

전송 매체

신호를 무선/유선으로 연결해주는 전달 경로입니다.

• 동축 케이블, 광섬유 케이블, 무선 전파 등

연결 장비

신호를 알맞게 변환하고 올바른 주소로 중계합니다.

• 공유기(라우터), 허브, 스위치 등

인터넷망에 연결된 서로 다른 기기들이 충돌 없이 패킷을 수발신하려면 각 노드에 유일한 숫자인 를 부여해야 합니다. 이는 32비트 체계인 IPv4와 주소 부족을 보완하기 위해 개발된 128비트 체계의 주소로 나뉩니다.

복잡한 숫자 주소를 사람이 일일이 외우기 힘들기 때문에 영문 텍스트 형태로 바꾼 명칭을 이라고 부르며, 브라우저에 영문 주소를 치면 이에 해당되는 실제 숫자 IP 주소로 자동 연계 및 조회 기능을 매핑해주는 변환 서비스를 (Domain Name System) 서버라고 합니다.

네트워크 세부 설정
1. (Subnet Mask)

IP 주소에서 네트워크 주소와 호스트 주소를 구별하는 필터입니다. 사용자가 속한 고유 서브넷 영역을 나누어 내부망과 외부망을 식별해 줍니다.

2. (Gateway)

로컬 네트워크에서 외부 네트워크로 이동할 때 거치는 '출입문' 장치(라우터) 주소입니다. 외부 인터넷망으로 향하는 패킷은 무조건 이 주소를 통과해야 합니다.

개념 2. 기기별 주요 네트워크 기술 (이더넷, 와이파이, 블루투스, NFC)

교과서 연계: I단원 유·무선 네트워크의 이해

[유선 네트워크 기술] 컴퓨터나 노트북에서 안정적인 유선 인터넷을 사용할 때 주로 탑재되는 표준 기술은 (Ethernet)입니다. 일반적으로 LAN 포트에 케이블을 직접 연결하여 외부 전파 간섭 없이 빠르고 일정한 인터넷 속도를 보장받습니다.

[무선 네트워크 기술] 전파나 적외선 등 공간을 매질로 데이터를 송수신합니다. 무선 공유기(AP)를 중심으로 일정 반경 내에서 무선 데이터 통신을 지원하는 (Wi-Fi), 무선 이어폰이나 키보드 연동을 위해 기기 간의 (Pairing) 절차가 필수적인 저전력 근거리 통신망 (Bluetooth), 그리고 10cm 이내의 매우 가까운 간극에서 리더기 시 데이터를 양방향 교환하는 교통카드·결제용 가 있습니다.

개념 3. 스마트팜과 사물인터넷(IoT)

교과서 연계: I단원 기술 발전과 생활

현대 정보 통신 기술의 핵심 융합 사례 중 하나인 스마트팜(Smart Farm)은 농가에 정보통신기술(ICT)을 접목하여 농작물과 가축의 재배 및 사육 환경을 스마트폰이나 컴퓨터로 원격 및 자동 제어하는 스마트한 농장입니다.

이를 작동시키는 기반 원천 기술이 바로 (IoT)입니다. 사물인터넷은 온·습도, 조도 등의 데이터를 실시간 수집하는 와 수집된 정보를 바탕으로 물을 뿌리거나 난방기를 작동시키는 환풍팬, 모터 등 명령을 물리적인 동작으로 이행하는 장치로 나뉩니다. 이 구성 요소들이 무선 네트워크망을 타고 연동하여 최적의 제어를 가능하게 만듭니다.

I. 컴퓨팅 시스템 단원 평가 연습 문제

네트워크 구성 요소 교과서 단원평가형

다음 중 유·무선 네트워크의 기본 구성 요소와 그에 속하는 장치가 올바르게 짝지어지지 않은 것은?

인터넷 주소 및 세부 체계 교과서 단원평가형

네트워크 주소 설계에서 IP 주소의 '네트워크 아이디'와 '호스트 아이디' 영역을 분리 필터링하는 정보와, 내부망을 벗어나 다른 외부 네트워크로 향할 때의 전송 경로 역할을 하는 장치의 명칭이 바르게 짝지어진 것은?

유·무선 통신 매체 및 기술 교과서 핵심형

컴퓨터나 스마트폰에서 주로 사용하는 네트워크 기술(이더넷, 와이파이, 블루투스, NFC)에 대한 설명으로 옳은 것만을 <보기>에서 고른 것은?

<보 기>
ㄱ. 이더넷(Ethernet)은 유선 랜선을 컴퓨터 포트에 직접 꽂아 외부 전파 간섭 없이 일정한 속도를 확보하는 유선 네트워크 표준 기술이다.
ㄴ. 와이파이(Wi-Fi)는 무선 공유기(AP)를 중심으로 일정 범위 내에서 무선 데이터 통신을 지원하는 기술이다.
ㄷ. 블루투스(Bluetooth)는 약 10cm 이내의 매우 가까운 거리에서 리더기 접촉식으로 동작하여 모바일 간편결제에 주로 활용된다.
ㄹ. NFC는 무선 이어폰, 키보드 등을 연동하는 기술로 10m 안팎의 거리에서 페어링 과정을 필수로 요구한다.
Chapter 02

II. 데이터

개념 1. 데이터 압축의 효과

교과서 연계: II단원 데이터 압축

데이터를 압축하면 크기가 줄어들어 자원 활용도가 극대화되며, 다음과 같은 3대 효과가 나타납니다.

1. 전송 속도 향상

전송되는 패킷 크기가 줄어들면서 최종 데이터의 가 비약적으로 빨라집니다.

2. 저장 용량 절감

하드웨어 및 클라우드 시스템의 물리적인 을 아껴 더 많은 자료를 보관할 수 있습니다.

3. 탄소저감 효과

네트워크 부하가 감소하여 전력 소비를 줄임으로써 친환경적인 효과에 기여합니다.

개념 2. 데이터 압축 방법

교과서 연계: II단원 데이터 압축

이미지와 소리 데이터는 정보 유실 여부에 따라 무손실 압축과 손실 압축으로 명확히 나뉩니다.

이미지 데이터 압축
A. 무손실 압축 :

동일한 색상 화소(픽셀)가 연속하여 중복 출력될 때, 그 값을 개별적으로 다 적지 않고 '연속된 픽셀의 개수'로 변환하여 파일 크기를 압축합니다. 화질 손상이나 데이터 왜곡이 전혀 없어 완벽한 원상 복구가 보장됩니다.

B. 손실 압축 : 다운 샘플링 & JPEG

인간의 눈이 세밀하게 구분해내기 힘든 미세한 색 영역을 누락하는 다운 샘플링을 진행한 후 대표적 손실 기법인 방식으로 포장해 압축률을 극대화합니다.

소리 데이터 압축
A. 무손실 압축 : 오디오

소리 파형을 표본화하여 부호화하는 전과정에서, 어떤 음향 신호도 임의로 누락시키거나 훼손하지 않고 데이터 기록 구조의 빈도 패턴만 압축하는 대표적 무손실 고음질 전용 기술 압축 방식을 취합니다.

B. 손실 압축 : 비가청 주파수 차단

사람이 귀로 물리적인 인식을 완벽하게 해낼 수 없는 20,000Hz 이상의 초음파 영역 등 비가청 대역에 속하는 가치 없는 소리 성분을 영구 삭제 및 간축시키는 방식으로 파일 용량을 줄이는 대표적인 방식이 있습니다.

개념 3. 암호화 방식 (블록 암호화, HTTPS, 파일 암호화, 생체 암호)

교과서 연계: II단원 데이터 암호화

네트워크와 시스템을 흐르는 정보를 지키기 위해 활용되는 대표적인 주요 암호화 기술들을 정리합니다.

1.

평문 메시지를 일정 크기의 묶음 단위인 '블록'으로 자른 뒤, 대칭키 방식을 사용해 각 블록마다 암호화 라운드를 거쳐 복잡한 암호문 구조를 생성하는 대칭키 핵심 기술입니다.

2. 프로토콜

웹 브라우저와 웹 서버 간 송수신 데이터 패킷의 탈취를 막기 위해, 공개키와 대칭키의 시너지를 결합한 SSL/TLS 암호 계층을 웹 통신 표준(HTTP)에 내장시킨 안전한 프로토콜입니다.

파일 암호화

컴퓨터나 모바일 기기 내부 저장 장치에 보관된 특정한 로컬 문서나 시스템 아카이브 파일 자체를 고유 암호화 알고리즘으로 시일링(Sealing)하여 인가된 비밀번호 없이는 파일 전개를 철저히 봉쇄하는 전용 보안 방식입니다.

3. (Biometrics)

복제가 거의 불가능한 개인의 지문, 홍채, 정맥 패턴 등 신체적 고유 특징을 일정한 수학적 해시 주소값으로 디바이스에 등록해 둔 후, 즉석 스캔 분석을 통해 단말 장치의 잠금을 안전하게 해제하는 첨단 보안 기술입니다.

개념 4. 빅데이터의 특징 (3V 체계)

교과서 연계: II단원 빅데이터의 이해

빅데이터(Big Data)는 수집 및 분석 가치를 도출해 내는 과정에서 다음과 같은 3V 특징을 보편적으로 가집니다.

1. 규모 (Volume)

기존 데이터 보관 및 처리 방식으로 감당하기 어려운 급 이상의 방대한 데이터 양을 특징으로 합니다.

2. 다양성 (Variety)

구조화된 수치 정보 중심의 데이터뿐만 아니라 사진, 영상, 텍스트처럼 형태가 고정되지 않은 다양한 데이터까지 모두 포함합니다.

3. 속도 (Velocity)

데이터의 생성 및 유통 주기가 매우 빨라짐에 따라 정보를 적시에 분석하여 반영하는 능력이 매우 중요하게 다뤄집니다.

개념 5. 데이터 시각화 방법

교과서 연계: II단원 데이터 수집과 시각화

방대한 데이터를 사람이 한눈에 인지하기 쉽도록 변환하는 3가지 대표적인 시각화 목적과 그 기법을 그림과 함께 분석합니다.

1. 특성 간의 관계를 나타내는 시각화

두 가짓수 이상의 변수들이 서로 가지는 밀접한 수학적 인과관계를 식별하기 위해 사용하며, 대표적으로 가로($x$)축과 세로($y$)축의 교차 지점에 데이터 관측치 점을 찍어 패턴을 파악하는 가 있습니다.

2. 시간의 흐름을 나타내는 시각화

시간의 경과나 연속적인 흐름에 따라 데이터 수치가 변화하는 일련의 양상을 분석하고 미래의 방향성을 예측하기 위해 사용됩니다. 대표적으로 각 데이터 포인트를 선분으로 이어 추세를 나타내는 가 널리 활용됩니다.

3. 값의 분포를 나타내는 시각화

개념 6. CSV 파일과 JSON 파일의 특징

교과서 연계: II단원 데이터 구조화

수집된 원본 정형/비정형 데이터를 시스템 간 호환성 높게 유통시키기 위해 대표적으로 사용하는 파일 형식은 다음과 같습니다.

CSV (Comma-Separated Values)

표 형태의 데이터를 저장하기 위해 사용하며, 각 데이터 필드를 로 구분하여 나타낸 일반 텍스트 형식입니다.

  • 구조가 매우 가볍고 단순하며 텍스트 뷰어로 즉시 열어볼 수 있습니다.
  • 용량이 아주 적고 호환성이 탁월해 데이터 수집 및 이식에 가장 널리 활용됩니다.

JSON (JavaScript Object Notation)

III. 알고리즘과 프로그래밍

개념 1. 오름차순 / 내림차순 정렬

교과서 연계: III단원 정렬 알고리즘

정렬(Sorting)은 뒤죽박죽 무작위로 섞여 있는 데이터셋을 기준 조건에 입각해 일정한 규칙 순서대로 재정렬하는 절차를 뜻합니다.

A. 오름차순 정렬 (Ascending)

값이 작은 데이터에서 시작하여 뒤로 갈수록 점차적으로 값이 위치하도록 나열하는 정렬 구조입니다. (예시: 1, 2, 4, 5, 8)

B. 내림차순 정렬 (Descending)

값이 가장 큰 대형 데이터가 맨 앞에 서고, 우측 뒤쪽으로 이동할수록 점진적으로 수치가 결합 배치되도록 설계하는 정렬입니다. (예시: 8, 5, 4, 2, 1)

개념 2. 선택 정렬 알고리즘 (Selection Sort)

교과서 연계: III단원 선택 정렬의 동작

선택 정렬은 정렬되지 않은 전체 영역을 탐색하여 가장 (오름차순 기준)을 찾아내어, 아직 정렬되지 않은 영역의 가장 첫 번째 원소와 자리를 교환하는 직관적인 정렬 알고리즘입니다.

리스트 [5, 1, 4, 2, 8] 선택 정렬(오름차순) 흐름 추적
- 초기 상태: [5, 1, 4, 2, 8] (미정렬 영역 전체 탐색 후 최솟값 '1' 식별)
- 1회전 후 : [1, 5, 4, 2, 8] (최솟값 '1'과 첫 번째 자리 원소 '5' 교환)
- 2회전 후 : [1, 2, 4, 5, 8] (나머지 중 최솟값 '2'와 미정렬 구역 첫 원소 '5' 교환)
- 3회전 후 : [1, 2, 4, 5, 8] (나머지 중 최솟값 '4' 탐색, 자기 자리이므로 유지)

개념 3. 버블 정렬 알고리즘 (Bubble Sort)

교과서 연계: III단원 버블 정렬의 동작

버블 정렬은 서로 두 개의 원소 크기를 비교하여, 지정한 정렬 조건에 맞지 않으면 서로 자리를 변경해 나가면서 정렬하는 원리입니다. 매 회전이 완료될 때마다 미정렬 구역의 가장 큰 값이 맨 뒤로 고정됩니다.

리스트 [5, 1, 4, 2, 8] 버블 정렬(오름차순) 흐름 추적
- 초기 상태: [5, 1, 4, 2, 8]
- 1회전 과정: (5,1) 비교교환 → [1,5,4,2,8] → (5,4) 비교교환 → [1,4,5,2,8] → (5,2) 비교교환 → [1,4,2,5,8] → (5,8) 유지
- 1회전 후 : [1, 4, 2, 5, 8] (가장 큰 원소인 '8'이 정렬 구역 맨 마지막으로 확정)
- 2회전 후 : [1, 2, 4, 5, 8] (그다음 큰 원소인 '5'가 해당 자리로 확정)

개념 4. 삽입 정렬 알고리즘 (Insertion Sort)

교과서 연계: III단원 삽입 정렬의 동작

삽입 정렬은 두 번째 원소부터 시작하여 정렬되지 않은 영역의 첫 데이터를 가져와, 앞쪽의 이미 정렬 완료된 구간의 데이터들과 뒤에서부터 비교하며 알맞은 제자리를 찾아 데이터를 정렬 알고리즘입니다.

리스트 [5, 1, 4, 2, 8] 삽입 정렬(오름차순) 흐름 추적 (시험지 수록 스냅샷)
- 초기 상태: [5, 1, 4, 2, 8] (두 번째 원소 '1'을 가져와 앞의 '5'와 대조하여 맨 앞에 삽입)
- 1회전 후 : [1, 5, 4, 2, 8] (세 번째 원소 '4'를 '1'과 '5' 사이의 올바른 칸에 삽입)
- 2회전 후 : [1, 4, 5, 2, 8] (네 번째 원소 '2'를 '1'과 '4' 사이에 삽입)
- 3회전 후 : [1, 2, 4, 5, 8] (다섯 번째 원소 '8'을 제자리에 삽입 정렬 완료)

개념 5. 탐색 알고리즘 (순차 탐색, 이진 탐색)

교과서 연계: III단원 탐색 알고리즘

데이터 더미 속에서 자신이 원하는 특정 값을 정확하게 조치 및 추출해 내는 과정을 탐색(Search)이라고 명칭합니다.

A. 순차 탐색 (Sequential Search)

데이터의 정렬 여부와 전혀 상관없이, 리스트의 맨 첫 번째 원소부터 출발하여 마지막 원소까지 하나씩 차례로 대조해 나가는 기초 탐색 방식입니다. 자료가 복잡하거나 많을 시 시간이 비효율적으로 소요될 수 있습니다.

B. 이진 탐색 (Binary Search)

반드시 데이터들이 오름/내림차순으로 된 자료 구조여야 작동하며, 비교 회전마다 중앙값과 대조하여 범위를 지속적으로 씩 줄여나가기 때문에 데이터가 많을수록 속도가 매우 빠르고 성능 면에서 대단히 우수합니다.

IV. 인공지능과 미래 사회

개념 1. 지능형 에이전트의 특성

교과서 연계: IV단원 지능형 에이전트

지능형 에이전트(Intelligent Agent)는 외부 환경을 스스로 인식하고 상황을 판단하여 자율적으로 동작하는 컴퓨팅 시스템입니다. 교과서에서 강조하는 지능형 에이전트의 6대 핵심 특성은 다음과 같습니다.

1.

사람의 직접적인 간섭 없이 스스로 수집된 환경 정보를 분석하여 동작과 행동 결정을 독자적으로 내리는 성질입니다.

2.

목표 달성을 위해 다른 에이전트나 인간(사용자)과 유기적으로 협력하고 긴밀히 소통하는 통신 능력을 의미합니다.

3.

외부 환경의 변화나 자극, 센서의 실시간 입력 값에 대하여 정밀하게 지체 없이 감지하고 대응하여 동작하는 성향입니다.

4.

환경에 피동적으로 수동 대응하는 것에 머무르지 않고, 스스로 목표를 세우고 먼저 주도적으로 행동을 개시하는 특성입니다.

5.

일회성 실행으로 종료되는 프로그램과 달리, 백그라운드 환경 등에서 계속 살아서 지속적·연속적으로 작동하는 성질입니다.

6.

새롭거나 낯선 미지의 환경에 직면하더라도, 현재 상황의 성격에 맞춰 유연하게 지식과 전략을 자발적으로 변모시키는 특성입니다.

개념 2. 인공지능의 발전 단계

교과서 연계: IV단원 인공지능의 역사와 발전

인공지능(AI)은 인간의 지능적 탐구 행위를 컴퓨팅 기기로 모방하는 학문으로, 발전 단계에 따라 다음과 같이 분류합니다.

A.

컴퓨터에 미리 짜놓은 규칙에 의해 답을 제공하는 방식입니다. 최초의 심리 치료 대화 봇인 엘리자(ELIZA)처럼 단순 매칭 규칙이나 명확한 문제 탐색 알고리즘을 사용했던 단계입니다.

B.

주체적인 자의식이 결여된 상태에서 특정한 한 영역(바둑, 기상 예측, 자율 주행, 번역 등)의 가이드 규칙 또는 축적 데이터에만 기초하여 인간 수준 이상의 효율을 증명해 보이는 오늘날 인공지능 기술의 주류 단계입니다.

C.

인간처럼 복합적인 이성적 사고와 자아 성찰, 주체적인 자의식을 탑재하여 영역에 제한받지 않고 스스로 상황을 유연하고 포괄적으로 이해하며 자발적으로 문제를 풀어낼 수 있는 이상적인 인공지능 미래형 단계입니다.

개념 3. 기계학습의 개념

교과서 연계: IV단원 기계학습의 이해

기계학습(Machine Learning)은 기계가 명시적이고 하드코딩된 규칙 설계 없이 데이터 자체로부터 패턴을 찾아 학습하는 기술입니다.

A.

컴퓨터에게 정답이 있는(정답 레이블이 제공되는) 훈련 데이터셋을 주고 기계를 훈련시키는 방법입니다. 이메일 스팸 여부 예측이나 주택 가격 전망 등에 필수적으로 활용됩니다.

B.

사전에 주어진 정답 레이블 없이 데이터의 고유한 속성과 패턴 특징을 기계가 자력으로 탐구하여 유사한 것들끼리 묶거나 구조적 군집을 찾아내도록 설계하는 학습 유형입니다.

개념 4. 기계학습의 분류 (분류, 회귀, 군집화)

교과서 연계: IV단원 기계학습의 세부 분류

기계학습은 당면한 해결 목표와 데이터 특성에 의거해 다음과 같은 3대 전용 기법으로 세분화됩니다. 각 기법의 개념적 도식 그림을 보고 알맞은 단어를 기입하세요.

1. (Classification)

입력 데이터를 이미 정의되어 있는 여러 개의 유한한 클래스 영역 중 하나로 구별하여 구분 짓는 지도학습 기술입니다. (예: 스팸 확인, 사물 식별)

2. (Regression)

입력 데이터셋을 기반으로 데이터 간의 연속적이고 지속적인 인과 수치 추세를 추론하고 특정 값을 수학적으로 연산 예측하는 지도학습 모델입니다. (예: 주가/연봉 예측)

3. (Clustering)

사전 정답 레이블 없이 데이터의 구조적 특징과 유사도만을 계산하여 자동으로 유사 집단(그룹)으로 가르고 결집시키는 대표적 비지도학습 기술입니다.

개념 5. 기계학습의 동작 과정

교과서 연계: IV단원 기계학습 적용과 과정

실생활 문제를 인공지능 기계학습을 사용하여 처리할 때 성립하는 고유한 기계학습 4대 설계 단계를 알맞은 단어로 작성하세요.

1단계 데이터

풀고자 하는 가치 있는 예측 문제를 모델링하기 위해 신뢰성 있고 목적에 꼭 부합하는 정형, 비정형 자료 및 정보를 사전에 확보하는 단계입니다.

2단계 데이터

수집한 자료 중 누락되었거나 중복되는 값, 신뢰를 저해하는 노이즈 오류를 찾아 보정·폐기하여 학습 모델 학습 시 오류를 최소화하도록 준비하는 정제 단계입니다.

3단계 모델 생성 및

정제된 최적의 기계학습 모델 알고리즘에 학습용 데이터를 통과시켜 수학적 규칙(패턴) 및 가중치 계수를 컴퓨터가 스스로 터득하도록 하는 연산 단계입니다.

4단계 모델 및 예측

학습 시 전혀 사용하지 않았던 검증 데이터(테스트 세트)를 투입해 예측값과의 정확도 및 정합성 오차율을 분석하고 실제 실생활 서비스에 탑재하여 운용하는 과정입니다.
